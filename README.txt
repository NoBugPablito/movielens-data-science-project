Pandas: Sirve para el analisis de datos, donde apoya en manipular, leer y analizar datos en tablas.
Numpy: Es una base matematica para poder realizar arreglos.
matplotlib: Para crear graficos y visualizaciones de los datos.
seaborn: Al igual que matplotlib ayuda a crear mejores graficos.
scikit-learn: Modelos de machine learning. 
sirprise: Sistema especial de recomendaciones, para construir modelos. 
streamlit: Para crear aplicaciones web interactivas.
